# getrish.github.io
Скрипт для начала установки RISH на новый сервер

Проверяет версию ОС и устанавливает необходимый минимум для начала установки RISH.
Поддерживаются AlmaLinux, Rocky Linux, RHEL и совместимые RHEL-системы.
Ubuntu/Debian и другие не-RHEL дистрибутивы не поддерживаются: RISH глубоко меняет настройки сервера, поэтому мы работаем только с предсказуемым RHEL-family окружением.

Если первичная настройка RISH уже выполнена, повторная команда установки не запускает `ri.sh`.
Для обновления или проверки RISH нужно запустить `mc`, нажать `F2` и выбрать пункт `Обновление/проверка RISH`.

Команда начала установки:

    curl -L get.rish.su | sh && /root/rish/ri.sh

Если у вас блокируются сервера РФ, то вот эта команда:

    curl -L getrish.sovmart.com | sh && /root/rish/ri.sh

Если на сервере не окажется установленной команды curl – ее можно установить с помощью 

    dnf install curl
